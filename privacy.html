Politique de confidentialité

Dernière mise à jour : 27 février 2026

Merci d'utiliser Email Outreach Assistant. Nous accordons une grande importance à la protection de vos données personnelles. Cette politique explique quelles informations nous collectons, comment nous les utilisons et quels sont vos droits.

1. Responsable du traitement

Le responsable du traitement de vos données est Said, joignable à said85000@gmail.com.

2. Données collectées

Notre application ne collecte aucune donnée personnelle sur nos serveurs. L'application fonctionne localement sur votre ordinateur. Les seules données traitées sont :

  • Fichiers importés : Les informations contenues dans les fichiers CSV ou Excel que vous importez (noms, emails, postes, entreprises, etc.). Ces données restent sur votre machine et ne sont pas transmises à nos serveurs.
  • API Gemini : Lors de la génération d'emails, les prompts contenant les données de vos prospects sont envoyés aux serveurs de Google pour traitement via l'API Gemini. Nous n'avons pas accès à ces échanges. Veuillez consulter la politique de confidentialité de Google pour plus d'informations.
  • Authentification Gmail : L'authentification se fait via OAuth. Nous recevons un jeton d'accès qui permet de créer des brouillons dans votre boîte mail. Nous ne stockons pas vos emails, ni vos identifiants. Le jeton est conservé localement sur votre ordinateur (fichier token.json).

3. Utilisation des données

Les données sont utilisées uniquement pour :

  • Générer des emails personnalisés via l'API Gemini.
  • Créer des brouillons dans votre compte Gmail (si vous le demandez).

Aucune autre utilisation n'est faite.

4. Partage des données

Nous ne partageons aucune donnée avec des tiers, sauf si la loi nous y oblige. Les échanges avec Google (API Gemini, OAuth) sont régis par leurs propres conditions.

5. Sécurité

L'application ne communique avec aucun serveur extérieur autre que les API Google que vous avez autorisées. Nous vous recommandons de protéger votre ordinateur et de ne pas partager votre jeton d'accès.

6. Vos droits

Vous avez le droit d'accéder, de rectifier ou de supprimer vos données. Comme nous ne stockons rien, ces droits s'exercent directement sur votre machine (suppression des fichiers importés, révocation de l'accès Gmail depuis votre compte Google).

7. Contact

Pour toute question, écrivez à saidcel99@example.com.
